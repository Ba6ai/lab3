## Задание 2



### Задача 1



### Текст задачи



#### Найти сумму тех элементов списка, которые начинаются на заданную цифру испольузя Seq.fold



### Алгоритм решения

###### 

#### В main: происходит запрос цифры с которой должны начинаться числа для суммы(target), далее в input происходит запрос чисел, проверка на завершение(Seq.takeWhile), Seq.choose - используется для проверки на ввод чисел, а не букв.
#### В Seq.fold в input - происходит проверка первого символа введённого числа(search) и в случает удачи складывает те числа, что начинаются на target

#### Отличие List.fold от Seq.fold в том что, List - работает только со списками и нагржает память, Seq - работает со всем что поддерживает интерфейс IEnumerable(массив, список, множ, строки) и не грузит память тк значения пролетают без сохранения, но с ними можно работать в момент пролёта.
